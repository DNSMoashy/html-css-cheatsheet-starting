HTML Tables Guide

Basic Table Info:

  • In HTML, a table is an element that allows for the representation of data in two dimensions; these dimensions are columns and rows.
  • Tables are composed of multiple smaller types of elements that have special meaning within the structure of the table. The elements that are used in the construction of a typical table are as follows:
  • Generally, tables are used to display tabular data in particular.
  • In modern development, the use of tables to position elements not semantically related to a table is an anti-pattern and should be avoided.

HTML Table Elements
tag(s) self closing? description
< table > no

Holds all table content with rows & columns.

Can NOT have both tbody & tr as direct descendants. One or the other.

< thead > no

Defines & contains the headings of columns within rows.
< tbody > no

More semantic element than tr. Contains all of the tables main content & tr elements that are not already in the header or footer.

< tfoot > no

table footer that uses table rows to summarize content at end of the table or make a footer.

< th > name

Used within the thead instead of td. Adds titles to the rows & columns of a table. MUST be enclosed within a tr.

< tr > table row

adds rows to a table before adding headings or data.

< td > table data

Nested inside table rows to add a cell of data

< tag rowspan = "" >, < tag colspan = "" > TABLE ATTRIBUTES
  • rowspan-indicates how many rows a cell should span
  • colspan-indicates how many columns a cell should span
