HTML Forms Guide

In HTML, forms are used for gathering and sending user input to an external source (like a server) for processing. This is done with the form tag, which contains various labels and inputs.

HTML Form Elements
tag(s) self closing? description
< form > no

Used to collect & send info to external source. Can contain various input elements that when submitted the info in the input is passed to the source named in the action attribute.

< input > yes

Used with type and value in conjunction with forms. info submitted to input sent to external source.

< text area > no

Used to create text box for multi-line input. The element supports rows and cols attr. which determine the initial size. Has both opening and closing tags; Can be stretched/shrunk by the user.

< select > no

Used to create dropdown list via an option element Only one option can be selected at a time. Selects name and Options value are a key value pair.

< label > no

Identifies a specific input based on matching ID & Value. Clicking label will focus field of related input.

< datalist > no

Pairing data list with input can make a basic search/autocomplete functionality. To do this both input and data list must have an ID that matches inputs value.

Data list stores options. Shown as a drop down when user clicks on input field.

< type = submit > Submittable Input

By making inputs type submit a submit button will be rendered to submit the form and execute its action.

Form Attributes
attribute input/type description
type = "checkbox" input

Renders single check box item. To create group use same name attribute. Multiple check boxes can be selected for same topic.

type = "number" input

Allow users to only enter #'s or special characters. When part of a form It will recieve key value pair of name : value.

type = "range" input

Creates a range slider as a selector between a min and max value attribute. Can be adjusted in increments with step attr. Meant as visual widget to adjust between values.

type = "text" input

Renders a single row input field users can type inside. Name & Value key value pair.

type = "radio button" Input

Renders single radio button. Can link multiple related buttons with the same name attribute. Only a single option can be chosen out of the group.

name = "" input name attribute

For a form to send data it must be in a key value pair. This is achieved via name attr. Name will be the "key".

type = "password' input

Renders single row of censored text within field.

required w/ in input

Specifies field must include value to be submitted. Put at end simply as required. no type needed.

max= & max length= w/ in input

Max attr. specifies max # value. Max length specifies maxiumum # of characters that can be entered in field.

pattern= "" w/in input

Uses regular expression to match against value of input. uses [].

min= & minlength= w/in input

Same but vice versa of max
