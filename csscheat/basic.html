Tips

  • Use classes when elements share many characteristics.
  • Use ID when element is unique
  • Keep Code: (D)on't (R)epeat (Y)ourself

CSS, or Cascading Style Sheets, is a language that is used in combination with HTML that customizes how HTML elements will appear.

CSS code can be Inline style or a separate file.

  • CSS styles can be directly added to HTML elements by using the style attribute in the elements opening tag. Each style declaration is ended with a semicolon.
  • More commonly CSS gets its own file to make code easier to read/maintain. The extension for CSS files is .css. These can be linked to an HTML file using a link tag in the head section.

To connect an external CSS file to its html file you would use the link element within the head tag. Normally included within the link element is:

  1. href attribute to specify the URL to the external resource
  2. rel attribute to specify the relationship of the linked document to the current document
  3. type attribute to define the type of content being linked

Anatomy

Rulesets: A ruleset is defined within either the HTML document itself, or in one or more separate files (known as stylesheets) that use the .css extension.

Next in the ruleset syntax is the block of declarations between the two braces {...}. This is the block of code that defines the style that will be applied. This is equivalent to the style attribute value in the inline-style.

The declaration block, or attribute value, is made up of one or more declarations. A declaration is made up of a CSS property name, followed by a colon (color:) then a value, followed by a semicolon, (blue;).

Selectors

CSS type selectors are used to match all elements of a given type or tag name. When using type selectors, elements are matched regardless of their nesting level in the HTML.

CSS can select elements by type, class, id, & attribute.

Specificity used when there are multiple conflicting property values that point to the same element. When determining which rule to apply, the selector with the highest specificity wins out. The most specific selector type is the ID selector, followed by class selectors, followed by type selectors.

Pseudo-Class: Allows elements to have to different states, such as hover or before and after click. Can be attached to any selector.

How To Single Selector:

  • *-Universal/All
  • .classname-by class
  • #idname-by ID
  • element- type selector; all like elements (ex. all p)
  • [attribute]-by attribute
    • EX: img[src*='']

How to Chain Multiple Selectors:

  1. Descendant Selector: match elements that are descended from one another
    • EX: div p {}
    • All p within div are selected. Parent THEN child/nested
    • Denoted by single space between each selector and descended selector.
    • All matching elements selected regardless of nesting.
  2. Chaining Selectors: selecting elements who also have the same class/ID, etc..
    • EX: h3.section-heading {}
    • All h3 elements with the class section-heading are selected.
    • Element and then selector WITHOUT a space.
  3. Grouping Selectors: Match multiple selectors to same CSS rule via comma
    • EX: h1, h2 {}
    • Separated by comma then space.
